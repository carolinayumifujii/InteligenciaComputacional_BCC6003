# Inteligência Computacional

Repositório destinado às atividades, experimentos e projetos desenvolvidos na disciplina de Inteligência Computacional.

---

## Sobre a Disciplina

A disciplina de Inteligência Computacional aborda métodos e técnicas inspiradas em processos naturais e heurísticas para resolução de problemas complexos, especialmente aqueles em que abordagens determinísticas tradicionais se mostram limitadas.

São explorados modelos computacionais capazes de aprender, adaptar-se e generalizar a partir de dados.

---

## Conteúdos Abordados

- Aprendizado de Máquina Supervisionado
- Árvores de Decisão
- Classificação e Regressão
- Validação e Avaliação de Modelos
- Heurísticas e Metaheurísticas
- Processamento e Análise de Dados
- Introdução a Redes Neurais (quando aplicável)

---

## Objetivo do Repositório

Este repositório tem como finalidade:

- Organizar implementações realizadas ao longo da disciplina
- Documentar experimentos com diferentes modelos computacionais
- Registrar análises comparativas entre abordagens
